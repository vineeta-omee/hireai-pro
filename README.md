# HireAI Pro

HireAI Pro is an AI-powered recruitment and hiring platform designed to simplify the hiring process for HR teams and recruiters.

## Features
- AI Resume Screening
- Candidate Management
- Hiring Pipeline
- Analytics Dashboard
- Email Automation
- Job Management
- Recruiter Dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- Supabase
- EmailJS
